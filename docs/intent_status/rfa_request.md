# 意图：rfa_request

## 标准槽位
- `unit_type`
- `count`
- `priority`
- `equipment`（可选列表）
- `window`（可选）

## 当前实现
- 仅定义 Schema，无处理逻辑。

## 待完成闭环
1. **业务规则确认**：资源/增援请求的审批流程、目标系统、数据格式。
2. **实现 Handler**：将请求写入业务库或调用外部调度平台。
3. **状态追踪**：返回请求编号，支持后续查询与撤销。
4. **权限校验**：确保只有有权限的用户能发起请求。

## 依赖与注意事项
- 与任务调度、资源管理系统联动，需提前确认接口。
- 注意与 `plan_task_approval`、`rescue-task-generate` 的协同关系。
