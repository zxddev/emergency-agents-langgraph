## ✅ 验证查询示例

### 查询1：指挥中心可控制的无人设备
```cypher
MATCH (c:RescueTeam {type: 'command_center'})-[:CONTROLS]->(e:Equipment)
RETURN e.name, e.type, e.specs
```
**结果**: 6个核心无人设备（翼龙-2H、DJI M350、FlyCart30、纵横CW-15、绝影X30、海豚3号）

### 查询2：能响应化工泄露的队伍
```cypher
MATCH (t:RescueTeam)-[:CAN_RESPOND_TO]->(d:DisasterEvent {type:'chemical_leak'})
RETURN t.name, t.specialization
```
**结果**: 3支专业化工救援队伍（攀枝花、泸州、德阳）

### 查询3：地震救援所需装备
```cypher
MATCH (d:DisasterEvent {type:'earthquake'})-[:REQUIRES]->(e:Equipment)
RETURN e.category, count(e) as count
ORDER BY count DESC
```
**结果**: 无人机11个、救援工具8个、工程设备7个、医疗设备7个...

### 查询4：成都重型救援队的装备清单
```cypher
MATCH (t:RescueTeam {name:'成都市重型救援队'})-[:OWNS]->(e:Equipment)
RETURN e.name, e.category, e.specs
```
**结果**: 8件装备（无人机、机器人、生命探测仪、液压破拆器、挖掘机、对讲机、急救包、发电机）

## 🔧 PostgreSQL查询示例

```sql
-- 查询2025年各类装备数量
SELECT category, COUNT(*) as count
FROM operational.equipment_2025
WHERE year = 2025
GROUP BY category
ORDER BY count DESC;

-- 查询指挥中心控制的无人设备
SELECT e.name, e.specs
FROM operational.command_unmanned_control_2025 c
JOIN operational.equipment_2025 e ON c.unmanned_equipment_id = e.id
JOIN operational.rescue_teams_2025 t ON c.command_center_id = t.id
WHERE t.type = 'command_center';

-- 查询能响应化工泄露的队伍
SELECT t.name, t.specialization, trc.capability_level
FROM operational.team_response_capability_2025 trc
JOIN operational.rescue_teams_2025 t ON trc.team_id = t.id
WHERE trc.disaster_type = 'chemical_leak'
ORDER BY trc.capability_level DESC;

-- 查询成都市重型救援队的装备清单
SELECT e.name, e.category, e.type, te.quantity
FROM operational.team_equipment_2025 te
JOIN operational.equipment_2025 e ON te.equipment_id = e.id
JOIN operational.rescue_teams_2025 t ON te.team_id = t.id
WHERE t.name = '成都市重型救援队';
```
